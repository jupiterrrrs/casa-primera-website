# Casa Primera Hotspring Resorts — Website

A resort booking website for Casa Primera Hotspring Resorts in Calamba, Laguna. Built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Responsive villa showcase with pricing and availability
- Dedicated rates section with per-villa package pricing
- Interactive booking form with a date-range calendar and live villa availability
- Guest reviews section
- Photo gallery
- Embedded location map
- Live chat widget

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS v4
- Radix UI / shadcn-style components
- Framer Motion (via `motion`)
- react-day-picker, date-fns

## Getting Started

Install dependencies:

```
npm install
```

Run the local dev server:

```
npm run dev
```

Build for production:

```
npm run build
```
